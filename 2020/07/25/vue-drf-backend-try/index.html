如果你先前从没接触过django，这段内容还是有难度的，我一步步讲解：

  1. modles.py文件用来定义模型，写django为我们封装好的代码，这里我们创建了一个Article类，并给他定义了title,body,main,created,updated，还定义了排序方式和在后台中的名称，并要求他在后台显示时返回某条目的标题。

  4. ·__str__表示在后台中返回的字段，这里返回title
